# NYC Health Inspection Letter Grade Chrome Extension

Chrome extension which displays the health inspection letter grade for restaurants in New York City on Yelp, Zagat, Seamless, Menupages, Foursquare and Opentable.  Shows the same card that restaurants display in their windows.

Unlike other extensions, this uses NYC's live data (updated daily). It also accounts for restaurants closed down by NYC Health Dept.

# Dependencies
Bluebird
